# 🎮 ppGames 🎮
 
A collection of lightweight browser games built using pure HTML5, CSS, and Vanilla JavaScript.
No frameworks. No external dependencies. Just simple and fun web games.
Each game is self-contained inside a single HTML file and runs directly in any modern browser.

### 🀄 ppEmoJong 🀄
**Status: 🔄 Early Access — playable, updates in progress**

https://ppistino.github.io/ppGames/ppEmoJong/ppEmoJong.html

A Mahjong Solitaire puzzle game with emoji tiles and a unique 3D progressive layout system.
Every board is mathematically guaranteed to be solvable.
Six game modes, an in-game shop, combo multipliers and power-ups keep every session fresh.

**Game Modes:**
- 🀄 Classic — match 2 identical tiles
- 🔱 Trio — match 3 identical tiles
- 💎 Quad — match 4 identical tiles
- ⚡ Arcade — new tiles fall every 12 seconds
- 🌸 Zen — no timer, play at your own pace
- 💣 Blitz — 60 second countdown

**Features:**
- Progressive 3D layouts: ~36 tiles at level 1 up to 144 at level 10+
- Stack height grows +1 layer every 3 levels (no ceiling — towers at high levels!)
- Shanghai-style scattered tile placement with random angles
- Guaranteed solvable boards via forward-construction algorithm
- Combo multiplier up to ×10
- Power-ups: ↩ Undo · 💡 Hint · 🔀 Shuffle · 💣 X-Ray Bomb · 🤖 AI Solver
- Special bonus tiles: 🪙 Coin group (+10🪙/tile) · 💎 Diamond group (+100🪙/tile)
- In-game shop with coin economy
- Save & Continue across sessions
- Game history (last 50 sessions)
- Pinch-to-zoom + zoom buttons
- Dark / Light theme
- Fully responsive — desktop and mobile

### 🚀 ppDeepOrbit 🚀
**Status: 🔄 Early Access — playable, updates in progress**

https://ppistino.github.io/ppGames/ppDeepOrbit/ppDeepOrbit.html

A multi-phase space mission game. Launch from Earth, blast through the asteroid belt,
land on alien planets to collect resources, and return safely home.
Earn credits on every mission and spend them on powerful upgrades for the next run.

**Mission Phases:**
1. 🌍 **Launch** — Thrust off the pad. Angle up to escape Earth's gravity.
2. ☄️ **Asteroid Belt** — Shoot through the belt. Collect resource drops ⚡💜🔩⚙️.
3. 🌑 **Planet Surface** — Land on the refuel pad. Collect terrain 🌿🪸🟡🍄 and ocean 🐚💧🦑 resources. Return once your quota is filled.
4. ☄️ **Return Journey** — Fight back through the belt with heavier enemy waves.
5. 🎯 **Landing** — Descend carefully. Stay vertical (±10°) and slow. Three outcomes: 🟢 Perfect · 🟡 Hard landing · 🔴 Crash.
6. 🛒 **Mission Complete** — Cargo auto-sold. Buy upgrades, then launch the next mission.

**Controls:**
- ⌨️ W / ↑ Thrust · A / ← Rotate left · D / → Rotate right
- ⌨️ S / ↓ Brake · Space / F Fire weapons
- ⌨️ 1–8 Activate upgrades · P Pause & save
- 📱 D-Pad · Fire button · Quick Shop · Upgrade buttons

**Upgrades (8 available):**
🛡️ Shield · 🔫 Double Cannon · ⚡ Ionic Engine · 🧲 Tractor Beam
🔋 Fuel Boost · 📡 Deep Radar · 🔧 Repair Kit · 💉 Medical Kit

**Features:**
- Multi-phase mission structure
- Resource collection and credit economy
- Progressive enemy waves (Fighters at lv3+, Satellites at lv5+)
- Upgrade shop between missions and mid-mission
- Fuel & Hull management
- Auto-save on pause
- Mobile D-Pad controls

### 🔢 ppSudoka Classic 🔢

https://ppistino.github.io/ppGames/ppSudoka/ppSudoka_Classic.html

A full-featured Sudoku game with four distinct game modes, all packed in a single HTML file.
Choose your mode, set the difficulty, and challenge your logic skills.

**Features:**
- 4 game modes — Classic · Visual (emoji symbols) · Irregular (free-form regions) · Killer (cage sums)
- 4 difficulty levels — Easy · Medium · Hard · Expert
- Notes mode, Undo, Check, Hints (max 3 per game)
- Score system with time, error and hint penalties
- Game history saved locally (last 50 sessions)
- Dark / Light theme toggle
- Fully responsive — works on desktop and mobile
