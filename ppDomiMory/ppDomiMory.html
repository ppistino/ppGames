❓ How to Play — ppDomiMory
🎯 Goal
Be the first to play all your tiles on the Domino board. You start by earning 7 tiles through the Memory phase. The AI starts with 7 tiles already dealt.
🧩 MEMORY Phase
A grid of face-down tiles appears (every tile has a duplicate). Flip two at a time — a match earns you that tile (+15 pts, +10 🪙). You need 7 pairs to start the game. During play, if you have no playable tile, you return here to find 1 new pair. The banner shows the board ends you need to match.
🎲 DOMINO Phase
Tap a green tile from your hand, then tap ◀ L or R ▶ to place it on the board. Each tile end must match the open board end. No match? → back to Memory to draw.
🏆 Scoring
Tile played: (a+b)×10 pts · Memory pair: +15 pts +10🪙
Win bonus: opponent pips ×5 pts · Block win: fewer pips = you win
🪙 Coins & Shop 🛒
Earn coins playing and matching. Spend them in the Shop — available from the Start Screen and the HUD during play. Helps carry over between games — unused items are saved automatically.
🆘 Memory Helps
🎯 Reveal — flips 4 random cards for 2 s
🔍 X-Ray — all cards faintly visible for 5 s
⚡ Auto-Match — auto-finds and flips any pair
🎲 Board Match — finds a pair matching the board ends
🕹️ Auto-Pilot — AI plays Memory for you (uses Board Match logic)
💥 Domino Power-ups
💣 Bomb — forces AI to draw 2 extra tiles on its next turn
🛡️ Shield — reduces your pip penalty when the game is blocked
⏳ Slow-Mo — doubles the Arcade timer for your next turn
⚡ Arcade Mode
Toggle on the Start Screen. A visible countdown bar runs during every turn — inside the game and inside the Memory panel.
🟢 Easy 30 s · 🟡 Medium 20 s · 🔴 Hard 12 s
Timer expired in Memory (initial draw): you exit with 9 tiles instead of 7.
Timer expired in Memory (mid-game draw): 2 extra tiles are added to your hand.
Timer expired in Domino: your turn is skipped.
A match in Memory resets the timer. Use Slow-Mo to double it!
🏁 After Each Game
▶ Continue — start the next game immediately (Arcade: timer −2 s, min 6 s)
↺ Retry — replay at the same level (Arcade loss only)
✕ Quit to Menu — return to the Start Screen
Your coins and unused helps are always saved.
