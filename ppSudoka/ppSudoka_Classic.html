❓ How to Play
🎯 Goal: Fill every cell with a number (1–9) so that each row, column and region contains all numbers exactly once.
⬜ CLASSIC Traditional 9×9 grid with 9 square 3×3 regions.
🎨 VISUAL Symbols instead of numbers! Animals, shapes, and icons. Perfect for visual memory. Each symbol maps to a hidden number 1–9.
🔷 IRREGULAR A 9×9 grid with free-form regions. Each game can feature colored organic regions or a maze-like layout with corridor walls—same logic, different look each time.
💀 KILLER Classic grid divided into "cages". The sum of numbers in each cage must match the shown total. No repeats within a cage! Fusion of Sudoku and Kakuro.
🛠 Tools: ✏️ Notes · ✓ Check · 🧹 Clear Notes · ↩ Undo · 💡 Hint (max 3)
