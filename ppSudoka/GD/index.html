❓ How to Play
🎯 Goal: Fill every cell with a number (1–9) so that each row, column and region contains all numbers exactly once.
⬜ CLASSIC Traditional 9×9 grid with 9 square 3×3 regions.
🎨 VISUAL Symbols instead of numbers! Animals and icons. Each symbol maps to a value 1–9.
🔷 IRREGULAR A 9×9 grid with free-form colored or maze-like regions.
💀 KILLER Grid divided into "cages". The sum in each cage must match the target. No repeats in a cage!
⚡ ARCADE An optional toggle you can activate on top of any game mode. A countdown timer appears below the progress bar — complete the puzzle before time runs out. Earn bonus coins based on remaining time and difficulty. If time runs out, watch an ad or spend coins to get +2 extra minutes!
🛠 Tools: ✏️ Notes · ✓ Check · 🧹 Clear Notes · ↩ Undo · 💡 Hint (limited)
🤖 AI Solver: Activate from HUD (🤖) to watch the virtual AI solve the puzzle step by step. No coins earned while AI is solving.
🪙 Coins: Earned by completing puzzles. Multiplied by difficulty and perfection bonus. Arcade mode adds a speed bonus. Spend coins in the Shop.
🛒 Shop: Buy extra Hints · extra Errors · unlock Pad Hints · auto-fill 3 cells · AI Solver (1 game) · +2 min Arcade time. Watch ads to earn more coins!
⌨️ Keyboard: Arrow keys to navigate · 1–9 to enter · Backspace to erase · N for note mode.
