❓ HOW TO PLAY

🎯 OBJECTIVE

Explore the hex planet, collect all required resources shown in the right panel, then reach the 🌀 PORTAL and press COLLECT to complete the level. Each level increases in size and difficulty.

🖱 CONTROLS

Tap/click a cell to select it. On desktop use keyboard shortcuts:

• R — Reveal selected cell

• C — Collect resource / use item

• F — Flag suspected mine

• +/- — Zoom in/out

• 0 / ESC — Reset view

• Arrows — Pan the map

• CTRL+D — Toggle Debug Mode (infinite resources)

⚡ ACTIONS

• 🔍 REVEAL — Uncovers the selected cell. Mine cells cost HP!

• ⛏ COLLECT — Gathers a resource or activates a found item.

• 🚩 FLAG — Marks a suspected mine. Flagged cells cannot be revealed accidentally.

⚠ MINES

Revealing a 💣 MINE costs HP. The number on each revealed cell shows how many mines are adjacent — use logic to find safe cells. Water hides 💣 WATER MINES too!

📦 RESOURCES

Land: 🌲 Wood · 🪨 Stone · ⚙️ Iron · 🪵 Coal · 🪙 Gold · 💎 Crystal · 🌿 Herbs · 🍄 Mushroom

Water: 🐠 Fish · 🦀 Crab · 🪸 Coral · 🫧 Pearl · 🐙 Octopus

New resource types unlock as you advance levels. Collect the exact amounts shown in the right panel.

✨ SPECIAL ITEMS

• 💥 BOMB — Reveals all neighbors (may hit mines for half damage)

• 🗺️ MAP — Reveals several random safe terrain cells

• 🧪 POTION — 60% chance to heal, 40% chance to damage

• 🔦 LANTERN — Safely illuminates all adjacent cells (no mine damage)

• 🩺 MEDKIT — Restores HP when collected

🛒 SHOP

Earn 🪙 coins by revealing cells and collecting resources. Spend them in the Shop to buy items or power-ups. Watch ads to get items for free! Prices increase with each purchase.

⏱ ARCADE MODE

Enable Arcade Mode from the start screen. A countdown timer is set based on the number of cells and resources — complete the level before time runs out! Use ⏳ Time Slowdown from the shop to double your remaining time.

🏆 SCORING

• +5 per revealed cell

• +20×level per collected resource

• +200×level for completing a level

• +coins for reveals & collects (used in Shop)
